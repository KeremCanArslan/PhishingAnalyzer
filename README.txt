Hazırlayan:Kerem ARSLAN  
Tarih:04.01.2026

Proje Tanımı
Bu uygulama, .eml uzantılı e-posta dosyalarını analiz ederek phishing tespit eden bir siber güvenlik aracıdır. 
Sistem; Header Spoofing, WHOIS/DNS sorguları ve Yapay Zeka (AI) yöntemlerini hibrit olarak kullanır.

Projenin çalışması için gerekli kütüphaneleri yüklemek için terminale şu komutu giriniz:

pip install -r requirements.txt
